Mates Rates CLI - CAB201

A CLI (command line interface) was to be written that would interract with three csv files.
The program was intended to work as an entry system for a car company, where the user could enter a new customer, rent a car to a customer, add, edit and remove vehicles, customer data, etc.

I did this project quite early on in my degree, so I wasn't particularly familar with the idea of a looping project - moreso building one that remains functional.
Other than that, this project felt quite trivial.

Each task was implemented in order after a basic input loop method, which would just be a prompt that returned values.
Class diagrams were written for the report along with a transcript.

There are several improvements that could be done, the main one which I would prefer is a nicer UI. CLIs have come a long way and these menus can be much nicer to print.
I will mention that the terminal clearing and entering values was extremely nice to do, and all criterias for this assessment were met.
